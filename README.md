# РАСПИС — мектеп кестесін автоматты құратын қосымша

## Іске қосу
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # өндірістік сборка (dist/)
```

## Не бар
- **Толық алгоритм** (src/algorithm/engine.ts): Greedy + Maximin, 18 қатаң ереже,
  шаршау формуласы, қос сабақ (×2), топ бөлу (Г1/Г2), спортзал ережелері,
  2 ауысым + ауысым аралық үзіліс, 10 стресс-тест, сапа есебі.
  Web Worker-де орындалады — UI қатпайды.
- **CRUD**: Сыныптар (оқу жоспары редакторымен), Мұғалімдер (5×8 шектеу
  сеткасымен), Кабинеттер (спортзал баптауымен), Пәндер (СанПиН базасы).
- **Нұсқалар**: сақтау, белсендіру, салыстыру (diff), Partial Generate.
- **Экспорт**: нақты Excel (.xlsx, әр сыныпқа бет) + басып шығару/PDF.
- **Сақтау**: барлық деректер localStorage-та (zustand persist) — бет
  жаңартылса да жоғалмайды.
- **Демо-деректер**: 22 сынып (1-11, екі ауысым), мұғалімдер, кабинеттер,
  оқу жоспарлары — бірден генерациялауға дайын. Қайтару: Алгоритм беті →
  «Демо-деректерге қайтару».
